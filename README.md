# 💳 Sistema de Caixa Eletrônico em C

## 📌 Sobre o Projeto

Este projeto é um sistema simples de caixa eletrônico desenvolvido em linguagem C.

O sistema permite:

- ✅ Depositar dinheiro
- ✅ Sacar dinheiro
- ✅ Consultar saldo
- ✅ Encerrar o programa

PATCH ATT 1:

- ✅ Modularizacao do codigo adicionada

O objetivo do projeto é praticar estruturas condicionais, variáveis e entrada de dados em C.

---

## 🛠 Tecnologias Utilizadas

- Linguagem C
- Compilador GCC
- Terminal / Prompt de Comando
